About This Project

Employee attrition is a significant challenge for organizations, leading to increased recruitment costs, loss of productivity, and decreased morale. This application uses machine learning to predict which employees are at risk of leaving and identifies the key factors contributing to attrition.

This project showcases the potential of machine learning in addressing HR challenges and improving employee retention strategies. Built with a Flask backend for machine learning, this application demonstrates how data science can be applied to solve real business problems.
